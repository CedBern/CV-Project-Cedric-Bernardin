# Portfolio & CV - Cédric Bernardin

## Description (FR)
Ce dépôt héberge le site portfolio et le Curriculum Vitae interactif de Cédric Bernardin, actualisé en mai 2026. Le site propose une expérience trilingue (français, espagnol, anglais) avec un commutateur de langue global en temps réel. Il présente son parcours professionnel d'enseignant de FLE, traducteur et concepteur pédagogique, ainsi qu'une grille détaillée de ses projets technologiques et d'économie circulaire.

### Fonctionnalités
- **CV Trilingue Complet** : Intégration de sa nouvelle expérience avec la plateforme *French Immersion* (par l'influenceur Carlito) et mise à jour de ses expériences académiques.
- **Grille de Projets Interactifs** : Fiches détaillées trilingues avec visuels haute fidélité pour :
  1. **MayaVoiceTranslator** : Outil de traduction vocale trilingue pour le maya yucatèque.
  2. **EcoValora México** : Plateforme intelligente de valorisation et recyclage des déchets solides.
  3. **Horizons** : Portail de webzines interactif (React + Vite) pour l'Alliance Française de Mérida.
  4. **MundialPronos** : Portail ludique de pronostics pour la Coupe du Monde de football.
- **Design Moderne & Réactif** : Interface conçue avec TailwindCSS, typographies dynamiques (Montserrat + Outfit) et floutages glassmorphismes.

## Description (EN)
This repository hosts the interactive portfolio and Curriculum Vitae website of Cédric Bernardin, updated in May 2026. The website provides a trilingual user experience (French, Spanish, English) with a global, real-time language switcher. It showcases his professional career as a French teacher, translator, and instructional designer, alongside a dynamic project grid presenting his technical and ecological initiatives.

### Key Features
- **Trilingual Resume**: Features his latest position as a Premium Oral Expression Coach with the *French Immersion* platform (by influencer Carlito) and updates former experiences.
- **Interactive Projects Dashboard**: Detailed trilingual sheets and high-fidelity mockups for:
  1. **MayaVoiceTranslator**: A trilingual Mayan-Spanish-French speech translator.
  2. **EcoValora México**: A smart solid waste recycling and circular economy management portal.
  3. **Horizons**: An interactive webzine portal (React + Vite) for the Alliance Française of Mérida.
  4. **MundialPronos**: A gamified football World Cup match predictions portal.
- **Modern Responsive UI**: Styled using TailwindCSS and custom CSS variables, optimized for all screen dimensions.
